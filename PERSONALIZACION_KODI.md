# Personalización de Kodi en Game Stick

## Lo que se puede modificar

- **Skin completo:** cambiar toda la interfaz (menús, colores, layout)
- **Menú principal:** agregar/quitar/reordenar items, poner accesos directos
- **Widgets:** contenido dinámico en el home (últimos vistos, favoritos, canales)
- **Fondos, colores, fuentes:** opciones dentro de cada skin
- **Acciones custom:** botones que ejecuten scripts (ej: toggle VPN)

## Skins recomendados (ligeros, para 256MB RAM)

| Skin | Peso | Nota |
|---|---|---|
| Estuary (default) | ~10MB | Ya viene instalado, funcional |
| Estuary MOD | ~12MB | Más opciones de menú, mismo rendimiento |
| Arctic Zephyr 2 | ~15MB | Bonito, limpio, moderado en recursos |
| Skin.Starter | ~2MB | Ultra minimal si hay problemas de RAM |

## Cómo instalar un skin

Settings → Add-ons → Install from repository → Look and Feel → Skin → Elegir → Install
